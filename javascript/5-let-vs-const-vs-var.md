# `let`, `const`, `var`

| Keyword | Scope    | Hoisted | Reassignable |
| ------- | -------- | ------- | ------------ |
| var     | Function | Yes     | Yes          |
| let     | Block    | Yes*    | Yes          |
| const   | Block    | Yes*    | No           |

\*Declarations are hoisted, but not initialized.
